## Proyecto Archivo Historico SDE
### Instrucciones al clonar el repositorio:

1. instalar Node.JS LTS
2. iniciar la carpeta en VS Code
3. instalar todas las dependencias con:
`npm install`
4. iniciar el proyecto con:
`npm run dev`
